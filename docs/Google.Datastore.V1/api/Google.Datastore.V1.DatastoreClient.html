Lookup(String, ReadOptions, IEnumerable<Key>, CallSettings)

ShutdownDefaultChannelsAsync()

Shuts down any channels automatically created by Create(ServiceEndpoint, DatastoreSettings) and CreateAsync(ServiceEndpoint, DatastoreSettings). Channels which weren't automatically created are not affected.

Declaration
public static Task ShutdownDefaultChannelsAsync()
Returns
Type Description
System.Threading.Tasks.Task

A task representing the asynchronous shutdown operation.
Remarks

After calling this method, further calls to Create(ServiceEndpoint, DatastoreSettings) and CreateAsync(ServiceEndpoint, DatastoreSettings) will create new channels, which could in turn be shut down by another call to this method.
