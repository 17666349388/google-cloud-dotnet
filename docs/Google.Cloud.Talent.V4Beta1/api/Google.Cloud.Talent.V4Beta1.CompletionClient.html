Create(ServiceEndpoint, CompletionSettings)

Synchronously creates a CompletionClient, applying defaults for all unspecified settings, and creating a channel connecting to the given endpoint with application default credentials where necessary. See the example for how to use custom credentials.

Declaration
public static CompletionClient Create(ServiceEndpoint endpoint = null, CompletionSettings settings = null)
Parameters
Type Name Description
ServiceEndpoint endpoint

Optional ServiceEndpoint.
CompletionSettings settings

Optional CompletionSettings.
Returns
Type Description
CompletionClient

The created CompletionClient.
Examples

This sample shows how to create a client using default credentials:

using Google.Cloud.Talent.V4Beta1;
...
// When running on Google Cloud Platform this will use the project Compute Credential.
// Or set the GOOGLE_APPLICATION_CREDENTIALS environment variable to the path of a JSON
// credential file to use that credential.
CompletionClient client = CompletionClient.Create();

This sample shows how to create a client using credentials loaded from a JSON file:

using Google.Cloud.Talent.V4Beta1;
using Google.Apis.Auth.OAuth2;
using Grpc.Auth;
using Grpc.Core;
...
GoogleCredential cred = GoogleCredential.FromFile("/path/to/credentials.json");
Channel channel = new Channel(
    CompletionClient.DefaultEndpoint.Host, CompletionClient.DefaultEndpoint.Port, cred.ToChannelCredentials());
CompletionClient client = CompletionClient.Create(channel);
...
// Shutdown the channel when it is no longer required.
channel.ShutdownAsync().Wait();

ShutdownDefaultChannelsAsync()

Shuts down any channels automatically created by Create(ServiceEndpoint, CompletionSettings) and CreateAsync(ServiceEndpoint, CompletionSettings). Channels which weren't automatically created are not affected.

Declaration
public static Task ShutdownDefaultChannelsAsync()
Returns
Type Description
System.Threading.Tasks.Task

A task representing the asynchronous shutdown operation.
Remarks

After calling this method, further calls to Create(ServiceEndpoint, CompletionSettings) and CreateAsync(ServiceEndpoint, CompletionSettings) will create new channels, which could in turn be shut down by another call to this method.
