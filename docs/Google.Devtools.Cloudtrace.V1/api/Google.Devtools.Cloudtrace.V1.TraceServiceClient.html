ListTraces(String, String, Nullable<Int32>, CallSettings)

Example:

TraceServiceClient client = TraceServiceClient.Create();
IPagedEnumerable<ListTracesResponse, Trace> traces = client.ListTraces(projectId);
foreach (Trace trace in traces.Take(10))
{
    Console.WriteLine($"{trace.TraceId}:");
    foreach (TraceSpan span in trace.Spans)
    {
        Console.WriteLine($"  {span.StartTime}-{span.EndTime}: {span.Kind}");
    }
}

ShutdownDefaultChannelsAsync()

Shuts down any channels automatically created by Create(ServiceEndpoint, TraceServiceSettings) and CreateAsync(ServiceEndpoint, TraceServiceSettings). Channels which weren't automatically created are not affected.

Declaration
public static Task ShutdownDefaultChannelsAsync()
Returns
Type Description
System.Threading.Tasks.Task

A task representing the asynchronous shutdown operation.
Remarks

After calling this method, further calls to Create(ServiceEndpoint, TraceServiceSettings) and CreateAsync(ServiceEndpoint, TraceServiceSettings) will create new channels, which could in turn be shut down by another call to this method.
