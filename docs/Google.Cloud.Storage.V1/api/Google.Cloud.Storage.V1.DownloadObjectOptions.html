Specifies whether or not the object's hash should be validated. If this property is null, the hash will always be validated.

Property Value
Type Description
System.Nullable<DownloadValidationMode>
Remarks

This is effectively an escape hatch for situations where hash validation fails. See https://github.com/GoogleCloudPlatform/google-cloud-dotnet/issues/1641 for background on this option. It is recommended that you leave this option unset unless you are knowingly downloading data for an object where hashing will fail.
