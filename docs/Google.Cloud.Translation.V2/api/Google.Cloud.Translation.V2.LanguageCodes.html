Class LanguageCodes

A helper class with constants for supported language codes. This is simply a code-convenient form of the set of languages returned by ListLanguages(String, Nullable<TranslationModel>). It is regenerated regularly, but not guaranteed to be complete at any moment in time; if the language you wish to use is present in the table but not covered here, please use the listed language code as a hard-coded string until this class catches up.
