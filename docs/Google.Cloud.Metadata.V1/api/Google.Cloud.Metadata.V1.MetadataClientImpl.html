public sealed class MetadataClientImpl : MetadataClient
Remarks

This is the "default" implementation of MetadataClient. Most client code should refer to MetadataClient, creating instances with Create(ConfigurableHttpClient) and Create(IHttpClientFactory). The constructor of this class is public for the sake of constructor-based dependency injection.
