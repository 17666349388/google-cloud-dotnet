public sealed class LogNameOneof : IResourceName, IEquatable<LogNameOneof>
Remarks

This resource name will contain one of the following:

  • LogName: A resource of type 'log'.
  • OrganizationLogName: A resource of type 'organization_log'.
  • FolderLogName: A resource of type 'folder_log'.
  • BillingLogName: A resource of type 'billing_log'.
