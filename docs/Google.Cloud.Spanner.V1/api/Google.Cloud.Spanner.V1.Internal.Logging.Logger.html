Class Logger

Provides diagnostic information on Cloud Spanner. By default, diagnostic logs will be sent to Trace.WriteLine for .Net desktop and stderr for .Net Core. The default LogLevel is LogLevel.Error which will only log error information. Performance logging can be turned on by setting LogPerformanceTraces = true.
