Remarks

The default OsLoginService scopes are:

  • "https://www.googleapis.com/auth/cloud-platform";
  • "https://www.googleapis.com/auth/cloud-platform.read-only";
  • "https://www.googleapis.com/auth/compute";
  • "https://www.googleapis.com/auth/compute.readonly";

Create(ServiceEndpoint, OsLoginServiceSettings)

Synchronously creates a OsLoginServiceClient, applying defaults for all unspecified settings, and creating a channel connecting to the given endpoint with application default credentials where necessary. See the example for how to use custom credentials.

Declaration

Parameters
Type Name Description
ServiceEndpoint endpoint

Optional ServiceEndpoint.
OsLoginServiceSettings settings

Optional OsLoginServiceSettings.
Returns
Type Description
OsLoginServiceClient

The created OsLoginServiceClient.
Examples

This sample shows how to create a client using default credentials:

using Google.Cloud.OsLogin.V1;
...
// When running on Google Cloud Platform this will use the project Compute Credential.
// Or set the GOOGLE_APPLICATION_CREDENTIALS environment variable to the path of a JSON
// credential file to use that credential.
OsLoginServiceClient client = OsLoginServiceClient.Create();

This sample shows how to create a client using credentials loaded from a JSON file:

using Google.Cloud.OsLogin.V1;
using Google.Apis.Auth.OAuth2;
using Grpc.Auth;
using Grpc.Core;
...
GoogleCredential cred = GoogleCredential.FromFile("/path/to/credentials.json");
Channel channel = new Channel(
    OsLoginServiceClient.DefaultEndpoint.Host, OsLoginServiceClient.DefaultEndpoint.Port, cred.ToChannelCredentials());
OsLoginServiceClient client = OsLoginServiceClient.Create(channel);
...
// Shutdown the channel when it is no longer required.
channel.ShutdownAsync().Wait();
