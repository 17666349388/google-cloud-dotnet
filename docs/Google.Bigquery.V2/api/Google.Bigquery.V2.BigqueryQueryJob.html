Remarks

If maxRows is smaller than the number of rows originally requested, this may require another server call to retrieve an appropriate page token. When working in pages, always specify a maximum per-request number of rows less than or equal to the page size you want.

GetRows()

Returns a lazy enumeration of all the rows in the query (from any originally-specified page token onwards), making server calls as required. The job must be completed before this method is called.
